# TopScrech.dev

## Setup

Make sure to install dependencies:

```bash
# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# bun
bun run dev
```

## Production

Build the application for production:

```bash
# bun
bun run build
```

Locally preview production build:
```bash
# bun
bun run preview
```

Check out the [deployment docs](https://nuxt.com/docs/getting-started/deployment) for more info
